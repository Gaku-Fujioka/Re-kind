# スクリーンショット

このディレクトリには、アプリケーションの主要画面のスクリーンショットを配置します。

## ファイル一覧

- `home.png` - ホーム画面のスクリーンショット
- `wallet.png` - ウォレット画面のスクリーンショット
- `calendar.png` - カレンダー画面のスクリーンショット

## 撮影手順

各画面の詳細な撮影手順は、各ファイルの `.md` ファイルを参照してください。

## 注意事項

- スクリーンショットは、デモデータが表示されている状態で撮影してください
- 個人情報が含まれないように注意してください
- 解像度は適切なサイズ（例: 1920x1080）で撮影してください
