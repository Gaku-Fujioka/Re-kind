# ウォレット画面スクリーンショット

## 説明

ウォレット画面では、保有しているKindCoinを確認し、寄付や花の贈与を行うことができます。

## 主要機能

- 保有KindCoinの表示
- 寄付機能（3つのプロジェクト）
- 花を贈る機能
- コインの使用履歴

## スクリーンショット

実際のスクリーンショットを `wallet.png` として配置してください。

**撮影手順**:
1. ユーザーBに切り替え（コインを保有している状態）
2. ウォレット画面を表示
3. 保有KindCoinが表示されている状態でスクリーンショットを撮影
4. `docs/wallet.png` として保存
