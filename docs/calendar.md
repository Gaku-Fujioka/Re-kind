# カレンダー画面スクリーンショット

## 説明

カレンダー画面では、日々の優しさの行動が花として記録され、連続記録日数（ストリーク）を確認できます。

## 主要機能

- 月次カレンダー表示
- 花の記録（daisy, tulip, lotus, sunflower, blossom）
- ストリーク表示
- 月の切り替え

## スクリーンショット

実際のスクリーンショットを `calendar.png` として配置してください。

**撮影手順**:
1. カレンダー画面を表示
2. 花が記録されている状態でスクリーンショットを撮影
3. `docs/calendar.png` として保存
